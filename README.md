## Quick Start - Get Trading in 5 Minutes

### 1. Clone & Setup
```bash
git clone https://github.com/yourusername/mithai-cap-trading.git
cd mithai-cap-trading
pip install -r requirements.txt
```

### 2. Launch the Platform
```bash
python app.py
```

### 3. Access Your Trading Terminal
- **Main Dashboard**: http://localhost:5000 - Market overview and summary
- **HFT Trading**: http://localhost:5000/hft - Professional trading interface
- **Advanced Charts**: http://localhost:5000/advanced_charts - Detailed analytics

### 4. Start Trading Like a Pro
- Select your preferred assets (NSE stocks, Bitcoin, Ethereum)
- Click "Start Bot" to begin automated market making
- Monitor real-time P&L and performance metrics
- Export data and generate comprehensive reports
